# Hack Club

Welcome to the **Hack Club** project! 🎉  
This is a simple web page built with **HTML**, **CSS**, and an image asset to introduce Hack Club and invite new members to join.

## 📖 About
Hack Club is a community of passionate individuals who love to code, create, and collaborate on exciting projects.  
Whether you're a beginner or an experienced developer, Hack Club provides a supportive environment to learn and grow your skills.

The webpage includes:
- A welcoming header and introduction
- Information about Hack Club activities (workshops, hackathons, networking events)
- A call-to-action button linking to [Hack Club Join Page](https://hackclub.com/join)
- An image (`image1.jpg`) to visually represent the club

## 🛠️ Technologies Used
- **HTML5** for structure
- **CSS3** for styling (`style.css`)
- **Image assets** (`image1.jpg`)
